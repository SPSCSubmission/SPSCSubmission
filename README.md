# Paper Text Materials

Here you can find all prompts, games and scenarios used in the "Can You Simulate Privacy Disclosure with LLM Agents in Conversations?" paper. 
It contains Markdown renderings of the prompts, scenario wording, and example model inputs used by the simulation framework.

## Files

- `prompt_templates.md`: reusable actor prompt templates and output-format instructions.
- `privacy_scenarios.md`: core privacy-disclosure scenario wording.
- `privacy_validation_battery.md`: focused privacy validation battery wording, contrast definitions, and formal annotations.
- `trust_game_scenarios.md`: trust-game scenario wording.
- `example_model_inputs.md`: example rendered system and user messages with sampled-profile placeholders.
- `observer_prompt.md`: post-hoc observer prompt and JSON output schema.

## Placeholder Convention

The live simulator samples a WVS/EVS profile and a Big Five profile for each agent. Those respondent-specific values are intentionally represented here as placeholders such as `[sampled WVS/EVS respondent values]` and `[sampled Big Five personality line]`.
